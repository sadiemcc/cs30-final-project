# Final Project Proposal

## Description
- A simple grocery store shopping game

## Needs to Have
- Movement 
- Store with shelving 
- Front/back/sides player model
- Interaction ability to grab items off shelves
- Self dialog (talk to yourself)
- Randomly generated grocery list
- Ability to hold up grocery list

## Nice to Have
- Running and stamina 
- Movement/idle animations
- Map of the store
- Sound effects
- Ghosts/a spooky factor 
- A storyline
- When interacting with a shelf, go first person
